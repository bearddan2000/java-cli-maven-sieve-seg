# java-cli-maven-sieve-seg

## Description
Find all primes less in a range. Follows functional programming practices.

## Tech stack
- java
- maven

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
